## Overview

**Law GPT** is an intelligent legal assistant designed to help lawyers, law students, researchers, and citizens with legal research, document analysis, case understanding, and drafting.

> ⚠️ **Disclaimer**: This tool is for informational and educational purposes only. It is **not a substitute** for professional legal advice from a qualified lawyer.

## Tech Stack

- **LLM**: GPT-4o / Grok / Llama 3 + Legal Fine-tuning
- **Framework**: LangChain / LlamaIndex
- **Backend**: FastAPI
- **Frontend**: React / Next.js + Tailwind CSS
- **Vector Store**: Pinecone / Chroma
- **Deployment**: Docker + Vercel / AWS
